Question-1: What are the different ways to select an element in the DOM?

In JavaScript, you can select DOM elements using different methods:

  • document.getElementById('id') - Selects an element by its ID.
  • document.getElementsByClassName('class') - Selects elements by class name.
  • document.getElementsByTagName('tag') - Selects elements by tag name.
  • document.querySelector('selector') - Selects the first element matching the CSS selector.
  • document.querySelectorAll('selector') - Selects all elements matching the CSS selector.

Question-2: What is the difference between innerHTML, innerText, and textContent?

These three properties are used to manipulate and retrieve text content from an element:

  • innerHTML - Returns or sets the HTML content inside an element
  • innerText - Returns only the visible text inside an element.
  • textContent - Returns all text inside an element, without parsing HTML.

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where a parent element handles events for its child elements. Instead of adding event listeners to multiple elements, you add a single event listener to a common ancestor and use event bubbling to capture events.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process in JavaScript where an event triggered on a child element "bubbles up" to its parent elements. This means that if an event occurs on a nested element, its ancestors will also receive the event unless stopped using event.stopPropagation().
